# FX Remit | The Instant Global Payment Layer

FX Remit 2.0 is a multi-chain remittance protocol designed to make global money transfers as fast and simple as sending a text message. By bridging decentralized finance with real-world banking, we enable users to send crypto directly to bank accounts and mobile wallets in over 20+ countries.

---

## Key Features

- Multi-Chain Native: Send assets from Base, Polygon, and Celo without complex bridging.
- Direct-to-Bank Payouts: Recipients receive local fiat (NGN, KES, GHS, BRL) directly in their bank accounts or mobile wallets.
- Ultra-Low Fees: Transparent 1.5% platform fee—significantly cheaper than legacy intermediaries.
- Verified Trust: Pre-flight account verification ensures your money always reaches the right person.
- Non-Custodial: Users maintain full control of their funds until the moment of exchange.

## Tech Stack

- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS & Vanilla CSS (styled-jsx)
- Protocol Integration: Paycrest Protocol API
- Deployment: Vercel (Edge Optimized)

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Fx-Remit/fx-remit-landingpage.git
   cd fx-remit-landingpage
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Open http://localhost:3000 to see the result.

## Repository Structure

- `/src/app`: Next.js pages and routing logic.
- `/src/app/components`: Reusable UI components (Hero, Features, Stats, Header).
- `/public`: Optimized assets and brand logos.
- `/src/app/globals.css`: Global CSS and Tailwind configurations.
